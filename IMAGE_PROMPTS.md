# Image Generation Prompts for CogniCare Specialist Pages

Use these prompts with an AI image generator (e.g., GPT DALL·E, Midjourney, or similar) to create header images. Save the generated images and add them to `src/assets/` as instructed.

---

## PECS Board Creator Header

**File to create:** `src/assets/pecs-header.png`

**Suggested dimensions:** 400×120px or similar wide format (will scale)

**Prompt:**
```
A minimal, modern illustration for the Picture Exchange Communication System (PECS). 
Show a friendly, inclusive visual with: colorful picture cards or icons arranged in a grid, 
a child’s hand exchanging a card, and a soft, encouraging atmosphere. 
Style: flat design, soft pastels (purple, blue, pink), clean lines, suitable for a healthcare/education app. 
No text. Warm and hopeful mood.
```

## TEACCH Tracker Creator Header

**File to create:** `src/assets/teacch-header.png`

**Suggested dimensions:** 400×120px or similar wide format (will scale)

**Prompt:**
```
A minimal, modern illustration for TEACCH structured teaching. 
Show: a visual schedule or work system (e.g., labeled bins, clear steps, organized spaces), 
perhaps a desk with materials in order, and a calm, structured environment. 
Style: flat design, warm oranges and greens, clean and organized, suitable for autism support and education. 
No text. Professional, supportive, and clear.
```
